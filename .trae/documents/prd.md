# 宝可梦 Elite Redux 2.65 · 地狱难度全属性通关网站 PRD

## 1. 产品概述

为《Pokémon Elite Redux v2.65》（基于绿宝石的高难度 GBA 改版）打造一份"地狱难度 + 全 18 属性通关"的硬核攻略与图鉴网站，帮助玩家理解 HELL MODE 的苛刻机制、按属性规划队伍、攻克每一名馆主与四天王。

- **目标用户**：ER 老玩家、打算挑战 HELL MODE 的硬核训练家、按属性通关（Monotype / All-Types Run）爱好者
- **核心价值**：把 v2.65 的 4 能力系统、72 起始宝可梦、HELL MODE 限制与 18 属性队伍搭配整合成可查询、可交互的攻略面板

## 2. 核心功能

### 2.1 用户角色
无注册体系，纯静态展示型网站，所有访客均为只读用户。

### 2.2 功能模块
1. **首页**：HERO 主视觉、ER 2.65 改版简介、HELL MODE 警示入口、四档难度对比、全 18 属性入口网格
2. **地狱难度详解页**：HELL MODE 全部 11 项机制说明、流程节点时间线（馆主→熔岩/海洋队→四天王→冠军）、关键限制可视化
3. **全属性通关页**：18 张属性卡片（火/水/草/电/冰/格斗/毒/地面/飞行/超能/虫/岩石/幽灵/龙/恶/钢/妖精/一般），每张含推荐宝可梦、4 能力搭配、关键招式、克制的馆主/天王

## 4. 用户界面设计

### 4.1 设计风格
- **整体基调**：暗黑地狱主题，致敬 GBA 像素质感，融合现代信息密度
- **主色**：地狱红 `#E11D2A`、烬黑 `#0A0A0C`、骨灰 `#1A1A1F`
- **副色**：熔岩橙 `#FF6B1A`、灵火蓝 `#3DD9FF`（用于冰/水/电等属性高亮）
- **18 属性色板**：完全沿用宝可梦官方属性配色（火=橙红、水=蓝、草=绿、电=黄…）
- **字体**：
  - 显示字体：`Press Start 2P`（GBA 像素感）用于主标题、数字
  - 标题字体：`Noto Serif SC` 中文衬线，增加东方"地狱绘卷"质感
  - 正文字体：`Noto Sans SC` 中文黑体
- **按钮**：硬边直角 + 内阴影 + 像素描边，悬停时熔岩橙发光
- **布局**：桌面优先，12 栏栅格 + 不对称排版，HERO 区域采用对角线分割
- **图标**：lucide-react 线性图标 + 自制 18 属性 SVG 像素徽章
- **动效**：火焰粒子背景、属性卡片悬停翻牌、滚动触发的灰烬飘落、能力徽章脉冲光

### 4.2 页面设计概览

| 页面 | 模块 | UI 元素 |
|------|------|----------|
| 首页 | HERO | 全屏 100vh，左下大标题"HELL · 18 TYPES"，右上角 ER 2.65 徽章，背景火焰粒子 canvas |
| 首页 | 难度对比 | 4 列对比卡，HELL 列描红边 + 闪烁警告条 |
| 首页 | 18 属性网格 | 3 行 6 列像素方格，hover 显示克制关系 tooltip |
| 地狱难度详解 | 机制清单 | 时间轴左侧 + 卡片右侧，每卡片含严重度星级 |
| 地狱难度详解 | 流程时间线 | 垂直时间轴，节点为馆主像素头像 |
| 全属性通关 | 属性轮盘 | 中央大徽章 + 左右切换器 |
| 全属性通关 | 推荐队伍卡 | 三联卡，含宝可梦名/4 能力徽章/关键招式标签 |
| 全属性通关 | 相克矩阵 | 18×18 热力图，红=被克、绿=克制 |

### 4.3 响应式
桌面优先（≥1280px 完整体验），平板（768-1279px）将 18 属性网格降为 3 列、HERO 字号缩小，移动端（<768px）单列纵向滚动、关闭火焰粒子以保证性能。

### 4.4 3D 场景指引
不适用，本站为 2D 信息型网站。
